RUN THESE COMMANDS IN YOUR TERMINAL:

cd /mnt/d/compliFi/dashboard
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps
npm start

This will:
1. Remove corrupted node_modules
2. Clean package-lock.json
3. Fresh install with correct dependencies
4. Start the dashboard

Expected output when it works:
"Compiled successfully!
You can now view complifi-dashboard in the browser.
Local: http://localhost:3000"
